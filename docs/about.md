这个博客聚焦四条长期主线：

- Java 后端基础
- 微服务与中间件
- AI 与大模型应用开发
- Vibe Coding 与开发提效

我希望它不是一堆零散文章的集合，而是一套可以持续扩展的技术知识库。这里会尽量保留“为什么这样做”的解释，而不只是结论或代码片段。

## 为什么做这个博客

一方面，我想把自己在 Java 后端、项目实战和 AI 应用开发中的学习过程沉淀下来；另一方面，我也想把对 Trae、Codex、Claude 这类 AI 工具的使用经验整理成可复用的方法论。

相比只追热点，我更关心两件事：

- 能不能把基础知识讲清楚
- 能不能把新方向和工程实践真正连起来

## 你会在这里看到什么

- 系统化的 Java 后端文章
- Spring、MySQL、Redis、微服务实践总结
- 大模型、RAG、工具调用、Spring AI 等 AI 工程内容
- 面向真实开发任务的 Vibe Coding 工作流与复盘
