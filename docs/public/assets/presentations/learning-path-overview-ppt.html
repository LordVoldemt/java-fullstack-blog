Key Section

为什么很多人学了很久，还是觉得自己不成体系

  • 常见原因通常有 4 个：
  • 看到热门内容就学，没有主线
  • 只输入，不输出
  • 学一个点就停一个点，知识之间没有连接
  • 总想一步到位，结果每一层都不扎实
04 / 08

Key Section

什么样的学习路线才算合理

  • 一个靠谱的学习路线，至少要满足这 3 个条件：
  • 1. 有主线
  • 你要知道自己当前阶段最重要的目标是什么，而不是被热点带着跑。
  • 2. 有层次
  • 先学基础，再学框架，再学工程化，再学架构扩展。不要把所有知识都摊成一层。
