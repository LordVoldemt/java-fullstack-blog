Key Section

Redis 为什么快

  • 很多文章提到 Redis，第一句话就是“Redis 很快”。这当然没错，但如果不继续问“为什么快”，这个认知就非常浅。
  • Redis 的快，主要来自几个方面：
  • 基于内存操作
  • 数据结构设计简单直接
  • 单线程模型避免了大量上下文切换和锁竞争
